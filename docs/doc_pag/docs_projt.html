Documentation

Welcome to Place

Place is an open-source platform that brings multiple AI models together in one place - fast, flexible, and designed to adapt to you. This documentation will help you get started with Place and make the most of its features.

Note

This documentation is a work in progress. Some features might change in future updates.

What is Place?

Place is a privacy-focused platform that allows you to run multiple open-source AI models locally on your device. It's perfect for developers and AI enthusiasts who want to experiment with different models without relying on external services.

Key Features

  • Run multiple AI models locally
  • Privacy-focused design
  • Customizable interface
  • Extensible architecture
  • Open source and community-driven

Installation

Follow these steps to install Place on your system:

Prerequisites

  • Python 3.8 or higher
  • pip (Python package manager)
  • Git

Installation Steps

# Clone the repository
git clone https://github.com/yourusername/place.git

# Navigate to the project directory
cd place

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py

Configuration

Place can be configured using the config.json file in the root directory. Here are the available options:

{
  "theme": "dark",
  "models_path": "./models",
  "default_model": "llama-2-7b",
  "max_tokens": 2048,
  "temperature": 0.7
}

AI Models

Place supports multiple AI models that you can use for different tasks. Here are some of the available models:

Llama 2

Meta's Llama 2 is a collection of pretrained and fine-tuned generative text models.

GPT4All

An ecosystem of open-source chatbots trained on a massive collection of clean assistant data.

Vicuna

An open-source chatbot trained by fine-tuning LLaMA on user-shared conversations.
